# ESERCIZIO

Chiedi quante ore si resta nel parcheggio. Se sono <= 2 ore costa 5€, se sono <= 5 ore costa 10€, altrimenti costa 15€.

## ESECUZIONE

- Leggi le ore indicate dall'utente
  - SE le ore indicate dall'utente sono minori o uguali a 2 ore stampa "Il costo è 5 €"
  - ALTRIMENTI SE le ore indicate dall'utente sono maggiori o uguali a 3 ore e minori o uguali a 5 ore stampa "Il costo è 10 €"
  - ALTRIMENTI SE le ore indicate dall'utente sono maggiori o uguali a 6 ore stampa "Il costo è 15 €"
