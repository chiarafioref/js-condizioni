# ESERCIZIO 

Chiedi l'età all'utente. Se è minorenne stampa "Ciao", se è over60 stampa "Salve", altrimenti stampa "Benvenuto".

## ESECUZIONE

- Leggi l'età indicata dall'utente
  - SE l'età indicata è minore di 18 stampa "Ciao"
  - ALTRIMENTI SE l'età indicata è maggiore di 60 stampa "Salve"
  - ALTRIMENTI SE l'età è maggiore di 18 e minore di 60 stampa "Benvenuto"
