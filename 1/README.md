# ESERCIZIO

Chiedi l'età dell'utente tramite prompt. Se l'utente è maggiorenne, stampa a console "Accesso consentito al locale".

## ESECUZIONE

- Leggi l'età indicata dall'utente.
- SE maggiore di 18 stampa "Accesso consentito al locale"
- ALTRIMENTI SE minore di 18 stampa "Accesso non consetito al locale"
