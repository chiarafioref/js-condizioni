# ESERCIZIO

Chiedi all'utente che giorno è oggi. Se è "sabato" o "domenica" stampa "Buon weekend!", altrimenti stampa "Buon lavoro".

## ESECUZIONE

- Leggi la risposta indicata dall'utente
  - SE l'utente indica "sabato" o "domenica" stampa "Buon weekend
  - ALTRIMENTI SE l'utente indica un altro giorno della settimana stampa "Buon lavoro"
